# flaskProject

### Project setup
```
pip install -r r.txt
```

### Compiles and hot-reloads for development on Windows
```
python app.py
```

## Compiles and hot-reloads for development on Linux
### run front
```
python3 app.py
```
### run background
```
nohup python3  -u app.py > run.log 2>&1 &
```
